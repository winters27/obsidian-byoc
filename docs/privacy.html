Privacy Policy

Last updated: April 25, 2026

Bring Your Own Cloud (BYOC) is an open-source Obsidian plugin that syncs your vault to your own cloud storage. Your privacy is a core design principle.

What we access

BYOC requests the drive.file scope from Google Drive. This means the plugin can only see, read, and write files and folders that it created itself. It cannot access any other files in your Google Drive.

Data storage

  • Your OAuth tokens are stored locally on your device, inside your Obsidian vault's plugin data.
  • No credentials, tokens, or vault data are transmitted to any third-party server.
  • The OAuth bridge page (this website) is a static redirect — it does not store, log, or transmit any data.

Data we collect

None. BYOC has no server, no analytics, no telemetry, and no tracking.

Third-party services

BYOC communicates directly with the Google Drive API using your own credentials. No intermediary services are used.

Open source

The full source code is available at github.com/winters27/obsidian-byoc.

Contact

For questions, open an issue on the GitHub repository.
